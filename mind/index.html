      # 思维

      # 深入思考

      # 训练方法

      1. 观察周围的现象，记录20处平日里习以为常的现象或者看到听到的结论，或者自己寻找问题，尝试深入地问自己为什么。
      2. 第一点提出的为什么问题中，每个给出至少3点思考的答案。
        例子：为什么微信会超过QQ的下载量？（现象）
        读书有什么用？（问题）
        为什么女性的方向感比男性弱？（结论）

      # 批判性思维方法

      # 1. 重设前提

      打破隐藏前提，重新设定前提进行思考，如玩游戏会影响学习成绩，前提是家长没玩过游戏，认为玩游戏是不听话的表现。实际是每个时代都有让人着迷的东西，以前是玩具，现在是游戏，本质没有变，变的是介质。

      # 2. 跨界诠释

      用其他领域的知识作为论据进行说明或者解释，得出新的观点，如经济学、信息学等。基于“万物相通”，注意两种能力：把简单的词汇抽象化和把抽象的概念通俗化。

      # 3. 改变视角

      同一件事情，从不同类人或者不同角度来看，性质是不同的，注意从多种角度去诠释。

      # 著名效应

      # 读书笔记框架

      读书，不能只当一个旁观者，只是被动接收信息，而要当一个交流者，与书本、与作者交流，学会思考和表达，和这本书建立联系。

      # 1. 阅读动机

      写下阅读动机，加强阅读动力，带着问题思考

      # 2. 核心概念

      实用类书籍一般围绕核心概念去论证，用自己的话解释这些概念，达到输入输出的目的

      # 3. 金句收集

      记录那些表达到位、思想深刻的句子，可以启发人的认知，增进理解，引发思考和激发动力

      # 4. 故事收集

      人不太容易记住概念，但是擅长记住故事，收集故事，思考故事背后的理念或者道理

      # 5. 发散联想

      举一反三，联想到类似的东西，一句话、一个新闻、自己的经历等等，建立关联进行记忆

      # 6. 洞见时刻

      当自己有恍然大悟的感觉时，记录这时的想法和感觉，加深对事物的认知

      # 7. 行动灵感

      书中涉及到的行为，想想自己的行动，试试能否将书中行动转化为自己的行动

      # 8. 阅读感受

      书中的内容，或是情绪表达、或是故事，触动自身时要学会记录和输出
